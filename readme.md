# Réalisation d'une vcard en HTML/CSS

## Description du projet
Ce projet a pour simple objectif de réaliser une vcard en utilisant les langages HTML et CSS. Celle-ci doit se focaliser sur une bonne utilisation de la sémentique pour HTML et l'utilisation des grids et flexbox pour le CSS. Il prévoit également la construction d'un repository Github idéal, incluant tout le contenu nécessaire à celui-ci.

## Roadmap
Voici les axes de travail qui constituent la roadmap de ce projet :
- [x] Composition du readme ;
- [x] Construction du repository et de ses dossiers/fichiers ;
- [x] Construction du vcard.html et du corps de la vcard en HTML ;
- [x] Récupération des images et couleurs voulues pour le CSS de la vcard ;
- [ ] Implémentation du CSS pour finaliser le rendu final ;
- [ ] Rendre la vcard responsive ;
